PROJECT WEB TOP UP GAME

project yang saya buat adalah sebuah website untuk top up game online. Website ini dirancang untuk memudahkan pengguna dalam melakukan pembelian voucher game dengan cepat dan aman. Pengguna dapat memilih berbagai jenis voucher game yang tersedia, melakukan pembayaran melalui berbagai metode, dan menerima kode voucher secara instan. Website ini juga dilengkapi dengan fitur keamanan untuk melindungi data pengguna dan transaksi.

PROJECT KONSTRIBUSI

konstribusi saya adalah menyambungkan setiap interface yang ada pada project ini. Saya juga bertanggung jawab dalam menguji setiap fitur untuk memastikan semuanya berjalan dengan baik. Selain itu, saya juga melakukan perbaikan bug yang ditemukan selama pengujian.

lalu membuat game offline di dalam website ini, sehingga pengguna dapat menikmati permainan tanpa perlu koneksi internet. Game ini dirancang untuk memberikan pengalaman bermain yang menyenangkan dan interaktif.
